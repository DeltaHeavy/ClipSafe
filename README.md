ClipSafe
========

**ClipSafe** clears your clipboard after you paste
so that passwords aren't just sitting around in there.

Written in a few hours. PRs welcome.

Requirements
------------
* `nodejs` with `npm`
* `npm install jpm`

Tested on
---------
* GitHub
* Facebook
* Amazon
* Google
* Yahoo
* Mozilla
* Coinbase
* Facebook
